# Azure Monitoring & KQL Queries

A collection of Kusto Query Language (KQL) queries used with Azure Monitor
and Log Analytics to analyze application performance and infrastructure health.

## Features
- Application Insights request analysis
- Azure Activity Log monitoring
- Virtual machine performance queries
- Real-world KQL patterns
